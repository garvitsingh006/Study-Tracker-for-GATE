# GATE 2027 Study Tracker

A personalized study tracker for GATE exam preparation (DA & CSIT). Track video lectures, notes, and PYQ practice across subjects with visual analytics.

## Tech Stack

- React 19 + Vite
- Tailwind CSS
- Recharts (charts & analytics)
- Lucide React (icons)

## Features

- **Exam Selection** — Choose between DA and CSIT streams
- **Dashboard** — Visual charts for study progress, streaks, and health scores
- **Subject Workspace** — Log video lectures, notes, and PYQ practice per subject
- **Streak Tracking** — Monitor daily study consistency
- **Data Persistence** — Local storage based, no backend needed

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
